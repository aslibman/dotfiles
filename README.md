# Dotfiles

Automated dotfiles management using [chezmoi](https://chezmoi.io) and [home-manager](https://github.com/nix-community/home-manager).

## Quick Start

```bash
sh -c "$(curl -fsLS https://raw.githubusercontent.com/aslibman/dotfiles/master/install.sh)"
```
